# Agent Health Monitoring Prototype

Interactive prototype for Agentforce Health Monitoring feature comparison between Beta and GA releases.

## Features

- **Beta Features (258-260)**: Core monitoring capabilities
- **GA Features (260+)**: Enhanced monitoring with additional features
- **Feature Comparison**: Side-by-side comparison with timeline filtering
